# 📋 Task Manager App – Java, JavaFX & MySQL

A simple yet functional Task Management Desktop Application built with Java, JavaFX, and MySQL, following the **MVC (Model-View-Controller)** architecture. This app allows users to efficiently manage their tasks through an intuitive graphical user interface built with JavaFX and styled using CSS.

---

## 🚀 Features

- 🔐 **User Authentication**  
  Secure Sign Up and Login functionality for users.

- 📝 **To-Do List Management**  
  Add new tasks to a personal to-do list.

- ✅ **Task Completion**  
  Move tasks from "To-Do" to "Completed" list.

- ✏️ **Edit Tasks**  
  Update task descriptions directly in the UI.

- 🗑️ **Delete Tasks**  
  Remove completed or unnecessary tasks permanently.

- 🎨 **Modern UI**  
  Built using JavaFX and styled with CSS for a clean, responsive design.

---

## 🛠️ Technologies Used

- ✅ Java (JDK 17+)
- ✅ JavaFX (UI framework)
- ✅ MySQL (Database)
- ✅ JDBC (Java Database Connectivity)
- ✅ OOP (Object-Oriented Programming principles)
- ✅ CSS (for custom styling)

---

## 🧱 Architecture

This application follows the **MVC architecture**:

- **Model** – Handles database connections and data logic  
- **View** – JavaFX FXML files and CSS for UI  
- **Controller** – Handles UI events, data passing, and logic between Model and View
